# Iris Flower Classification using Flask and KNN

## Overview

This project is a Machine Learning web application built using Flask and Scikit-Learn. It predicts the species of an Iris flower based on four input features:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

The model uses the K-Nearest Neighbors (KNN) algorithm trained on the Iris dataset.

## Features

* Interactive Web Interface
* Real-time Flower Prediction
* Machine Learning using KNN
* Flask Backend
* Responsive UI Design

## Technologies Used

* Python
* Flask
* Scikit-Learn
* HTML
* CSS

## Dataset

The Iris dataset contains three flower species:

1. Setosa
2. Versicolor
3. Virginica

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/iris-flower-classification.git
cd iris-flower-classification
```

Create a virtual environment:

```bash
python3 -m venv myenv
source myenv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python3 flower_app.py
```

Open your browser and visit:

http://127.0.0.1:5000

## Project Workflow

1. User enters flower measurements.
2. Flask receives the input.
3. KNN model predicts the flower species.
4. Result is displayed on the webpage.
